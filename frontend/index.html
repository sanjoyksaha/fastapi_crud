This Indoor Navigation System is designed to streamline and automate indoor routing and task management within complex facilities. Equipped with a user-friendly interface, the system offers robust access and task automation features:

Authentication System

Secure login with user authentication ensures that only authorized personnel have access to system features and sensitive data.

SuperAdmin User Management

A dedicated SuperAdmin role allows for comprehensive control over user management. SuperAdmins can add new users, assign roles, and monitor access levels, ensuring a secure and organized user structure.

User Login with ID

Registered users can conveniently log in with their unique IDs, simplifying access and enhancing individual accountability within the system.

Robotic Assistance

Users can summon a robot to perform specific tasks within the facility. The robotic assistant improves operational efficiency by taking on designated jobs, reducing manual workload, and enhancing response times.

Log History Access

Users can view their activity history, including job requests, navigation routes, and system interactions. This feature offers transparency and assists users in tracking their actions for improved accountability and productivity.
